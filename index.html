An editable <textarea> that supports any syntax highlighting algorithm, for code or something else. Also, added plugins.

Aiming to be more flexible, lightweight, modular, progressively enhanced and standards-based than the alternatives, we support HTML forms, the <textarea> JavaScript interface, more languages and more use cases, but don't expect a heavyweight preconfigured IDE - you need to select the plugins you want first or use an alternative.

By using code-input-js and any syntax-highlighting library of your choosing, you can create editable, syntax-highlightable, customisable elements that look like <textarea>s to HTML forms and JavaScript code, look like normal non-editable <pre><code> blocks to the syntax-highlighting library, and have a convenient plugin API plus existing plugins for commonly-wanted functionality.

Many Plugins Provided, or Contribute your Own!

Nevertheless, if code-input doesn't suit you and you want a fully-featured, packaged but likely heavier code editing component, and don't need compatibility with different highlighters, HTML forms or textarea APIs, we recommend something like CodeMirror, Ace or Monaco.
